# Docker PHP-FPM Nginx

This is a Docker image for running a PHP-FPM server with Nginx as a reverse proxy.

## Features

- PHP 8.2
- PHP-FPM
- Composer
- Nginx
- Supervisor
- PostgreSQL and MySQL support
- Xdebug support
- Based on Alpine Linux
